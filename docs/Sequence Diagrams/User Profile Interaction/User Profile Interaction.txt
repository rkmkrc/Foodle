title Foodle User Profile Interaction Sequence Diagram

participant User

participant UI

participant Server

participant Database Server

loop 
User->UI: showUserProfiles()
UI->UI: searchUser(userFilter)
UI->Server: getUsers(userFilter)
Server->Database Server: getUsers(userFilter)
Database Server-->Server: users
Server-->UI: users
end

loop
User->UI: selectUser()
UI->Server: getUserInfo(userID)
Server->Database Server: getUserInfo(userID)
Database Server-->Server: userInfo
Server-->UI: userProfile
end 

loop
User->UI: followUser()
UI->Server: postUserFollowRequest(userFollowBody)
alt followRequest
Server->Database Server: sendUserFollow(userID, userIDToFollow)
else unfollowRequest
Server->Database Server: sendUserUnfollow(userID, userIDToUnfollow)
end

Database Server-->Server: followResult
Server-->UI: followResult
end
